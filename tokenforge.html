Token Forge

デザイントークン自動最適化ツール
@akiy2009/token-forge

デザイントークンを自動で最適化し、 背景色に合わせたコントラスト調整・色相/明度/彩度の微調整により アクセシブルで一貫した UI カラーを生成する軽量ツール。

npmで見る

✨ 特徴

🎨 WCAG AA / AAA に対応したコントラスト自動調整
🔧 HSLベースの色相・明度・彩度の補正
🏷 JSON / CSSトークンの自動正規化
⚡ CLIツールとしてすぐ使える
🧩 PostCSSプラグイン対応
📦 依存ほぼゼロの軽量設計

📦 インストール

npm install @akiy2009/token-forge

🚀 CLI 使用例

JSONトークン最適化

adt-optimize --input tokens.json --output tokens.fixed.json

CSS変数最適化

adt-optimize --input styles.css --output styles.fixed.css --format css

🗂 JSONトークン例

入力

{
 "primary": "#4c8cff",
 "background": "#ffffff",
 "success": "#28c76f"
}

出力（例）

{
 "primary": "#2d6fe8",
 "background": "#ffffff",
 "success": "#219c58"
}

⚙ PostCSS プラグイン

module.exports = {
 plugins: [
   require("@akiy2009/token-forge/postcss")({
     targetContrast: 4.5,
     adjustHue: true,
     adjustSaturation: true
   })
 ]
};
