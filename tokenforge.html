Token Forge

デザイントークンを自動最適化し アクセシブルで一貫した UI カラーを生成するツール

@akiy2009/token-forge
npmを見る GitHub

✨ 特徴

🎨 WCAG AA / AAA コントラスト調整
🔧 HSL 色補正アルゴリズム
🏷 JSON / CSS トークン対応
⚡ CLIツール
🧩 PostCSSプラグイン
📦 軽量設計

📦 インストール

npm install @akiy2009/token-forge

🚀 CLI 使用例

JSONトークン

adt-optimize --input tokens.json --output tokens.fixed.json

CSS変数

adt-optimize --input styles.css --output styles.fixed.css --format css

🗂 JSONトークン例

入力

{
 "primary": "#4c8cff",
 "background": "#ffffff",
 "success": "#28c76f"
}

出力（例）

{
 "primary": "#2d6fe8",
 "background": "#ffffff",
 "success": "#219c58"
}

🧩 PostCSSプラグイン

module.exports = {
 plugins: [
   require("@akiy2009/token-forge/postcss")({
     targetContrast: 4.5,
     adjustHue: true,
     adjustSaturation: true
   })
 ]
};
