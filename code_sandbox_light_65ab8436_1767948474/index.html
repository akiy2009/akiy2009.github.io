UKKについて

香川高専 宇宙開発研究部（UKK）は、宇宙開発に興味を持つ学生が集まり、 実践的な研究開発活動を行っている部活動です。

気象観測機器の開発、CanSat（模擬人工衛星）の製作、 ロケット開発など、多岐にわたるプロジェクトに取り組んでいます。

活動内容

気象観測機器開発

温度、湿度、気圧などのセンサーを用いた気象観測システムの設計・製作を行っています。 全国高等学校・高専気象観測機器コンテストへの出場を目指しています。香川高専は常連出場校です。

講座

空き缶サイズの模擬人工衛星を開発し、上空から落下させながら ミッションを遂行するプロジェクトに取り組んでいます。

KOSEN衛星製作

全国10高専が参加するKOSEN衛星プロジェクトに参加し、 衛星の設計・製作・運用に関する技術を学んでいます。

電子工作

Arduino、Raspberry Piなどのマイコンを使用した 電子回路の設計とプログラミングのスキルを磨いています。

プログラミング

C/C++、Pythonなどを用いたデータ処理、制御プログラム、 シミュレーションソフトウェアの開発を行っています。

勉強会

宇宙工学、電子工学、プログラミングに関する 定期的な勉強会を開催し、知識と技術を共有しています。
