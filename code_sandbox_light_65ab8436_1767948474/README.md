# 香川高専 宇宙開発研究部（UKK）公式サイト

## 📖 プロジェクト概要

香川高等専門学校 宇宙開発研究部（Uchu Kaihatsu Kenkyubu - UKK）の公式Webサイトです。最新のWeb技術を使用し、モダンで美しいデザインとスムーズなユーザー体験を提供します。

## ✨ 主な機能

### 完成済み機能

1. **レスポンシブデザイン**
   - デスクトップ、タブレット、モバイルに完全対応
   - 画面サイズに応じた最適なレイアウト

2. **モダンなUIデザイン**
   - 宇宙をテーマにしたダークモードデザイン
   - グラデーション、アニメーション、パーティクル効果
   - ガラスモーフィズムとニューモーフィズムの要素

3. **インタラクティブ機能**
   - スムーズなスクロールアニメーション
   - ページ内ナビゲーション（スムーススクロール）
   - スクロールに応じた要素の表示アニメーション
   - プログレスバーのアニメーション
   - 統計カウンターのアニメーション

4. **コンテンツセクション**
   - **ホーム**: 印象的なヒーローセクション（星空エフェクト付き）
   - **UKKについて**: 部活動の紹介と統計情報
   - **活動内容**: 6つの主要活動の詳細
   - **プロジェクト**: 進行中プロジェクトのステータス表示
   - **実績**: タイムライン形式の実績紹介
   - **メンバー**: メンバー紹介と入部案内
   - **お問い合わせ**: コンタクトフォームと連絡先情報

5. **ナビゲーション**
   - 固定ヘッダーナビゲーション
   - モバイル用ハンバーガーメニュー
   - アクティブセクションのハイライト表示
   - トップに戻るボタン

6. **フォーム機能**
   - バリデーション機能付きお問い合わせフォーム
   - リアルタイムフィードバック（通知システム）

## 🔧 カスタマイズ方法

### 1. カラースキームの変更
`css/style.css`の`:root`セクションでCSS変数を編集：

```css
:root {
    --primary-color: #0066ff;
    --secondary-color: #00d4ff;
    --accent-color: #ff3d00;
    /* その他の変数 */
}
```

### 2. コンテンツの編集
`index.html`の各セクションで内容を変更できます。

### 3. アニメーションの調整
`js/main.js`の`observerOptions`やアニメーション設定を変更。
